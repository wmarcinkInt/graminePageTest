gramine-sgx-quote-view – Display SGX quote structure¶

Synopsis¶

gramine-sgx-quote-view [OPTION]… FILE

Description¶

gramine-sgx-quote-view displays internal structure of an SGX quote contained in FILE.

Command line arguments¶

-h, --help¶

Display usage.

-m, --msb¶

Print hex strings in big-endian order.
